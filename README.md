**A program which supports conversion from MIRAX (.mrxs) images to SVS (.svs) images for WSI-related tasks**

Installation and use:
1. Make sure you have the required installations for running code available (Python, PyPI for package installation, etc.)
2. Clone the repository from GitHub
3. Drag the MIRAX file(s) you need to convert into the repo directory. Make sure to include all data files related to the MIRAX image.
4. Run the run.bat file and enter the name of the image you want to convert. The resulting SVS image will be put into the "output" folder.
